# stock-monitor 股票监控技能

个人股票持仓监控助手。实时查询股票现价，自动计算距离买卖点的百分比，直观展示当前持仓状态，辅助判断买卖时机。

## 触发关键词

- **监控类**：股票监控、看看股票、股票报告、持仓状态、自选股状态
- **查询类**：查一下股票、股票行情、股价多少、现在股价
- **状态类**：买入点、卖出点、涨跌幅度、距买点、距卖点
- **时间类**：早盘、午盘、收盘、每日股票

当用户表达查看股票行情或了解持仓状态的意图时触发此技能

## 执行步骤

1. 运行脚本获取股票数据：
   ```
   python3 /Users/ricardo.chen/.qclaw/skills/stock-monitor/fetch_stocks.py
   ```

2. 将输出结果**直接返回给用户**，不做任何总结或额外解读
